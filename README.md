This project automates functional test scenarios for the **Zara.com** e-commerce website using **Java, Selenium, and TestNG**.  
It follows the **Page Object Model (POM)** design pattern for maintainability and reusability.  

Main objectives:
- Navigate through Zara web pages  
- Select and open product categories  
- Add a random product to the shopping cart  
- Validate product details and prices  
- Handle dynamic elements and modal pop-ups
